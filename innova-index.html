🗄️ Reset Product Catalogue
Wipes all products from the database and replaces them with the default Innova Group catalogue. Bookings are not affected.

Permanently removes every product from the database. Does not affect bookings. Use this to start the catalogue from scratch.

Permanently deletes every booking from the database and clears all locally-stored client information. This cannot be recovered.
